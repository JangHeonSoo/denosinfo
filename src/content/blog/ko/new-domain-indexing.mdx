글은 꾸준히 쓰고 있다.  
색인도 조금씩 늘어나는 것 같다.  
근데 구글에서 들어오는 사람 수는 아직도 0.

처음 블로그 만들면 거의 다 겪는다.  
나만 유령 사이트 된 거 아닌가 싶지만,  
데이터를 보면 <strong>“어느 정도의 0”</strong>은 그냥 정상 구간이다.

![희망찬 아침 햇살이 비치는 차분한 워크스페이스와 0명의 방문자 데이터](/images/new_domain_hero_waiting_game_1770944226195.png)

<br/>

## 이 글에서 다루는 것

- 완전 신규 도메인에서 “검색 유입 0”이 흔한 기간
- 몇 달까지는 걱정보다 관찰 모드로 두면 되는지
- 0 구간에서 뭘 하고 있어야, 나중에 한 번에 튀는지

<br/>

## 1. 색인과 유입은 완전히 다른 타임라인

먼저 이거부터 구분해야 한다.

- <strong>색인</strong>:  
  - 구글이 페이지를 찾아서 인덱스에 넣는 단계  
  - 빠르면 몇 시간~며칠, 느리면 몇 주까지도 걸린다.

- <strong>검색 유입(트래픽)</strong>:  
  - 그 페이지가 실제 검색 결과에 어느 정도 순위로 노출되는 단계  
  - 보통 <strong>몇 달 단위</strong>로 봐야 한다.

경험치/연구들을 대충 정리하면:

- 새 페이지가 인덱스에 처음 잡히는 건 4–7일 정도가 <strong>“평균적인 케이스”</strong>라는 얘기가 많고
- 눈에 보이는 순위·유입 변화는 보통 1–3개월 이후부터 슬슬 나타나기 시작한다.

그러니까 <strong>“색인은 됐는데 유입이 0”</strong>인 상태 자체가  
생각보다 아주, 아주 흔한 구간이다.

<br/>

## 2. 신규 도메인에서 0이 유지되는 “평균 구간”

신규 도메인 기준으로, 여러 자료와 실전 후기를 섞어보면 대략 이런 느낌이다.

- <strong>0–1개월차</strong>  
  - 색인이 일부만 됐거나, 아예 안 된 상태도 많음  
  - 검색 유입 0, 혹은 하루 1클릭도 안 되는 게 거의 당연

- <strong>1–3개월차</strong>  
  - 저경쟁 키워드나 아주 마이너한 글이 조금씩 인상/impression을 받기 시작  
  - 트래픽은 <strong>“하루 1–5 클릭 왔다 갔다”</strong> 수준이면 선방하는 편

- <strong>3–6개월차</strong>  
  - 제대로 세팅된 사이트라면 이쯤에서 키워드 움직임이 눈에 띄게 늘기 시작  
  - 많은 SEO들이 <strong>“새 도메인은 최소 4–6개월은 봐야 뭔가 보인다”</strong>고 말함

- <strong>6–12개월차</strong>  
  - 제대로 운영했다는 전제하에, 이 시기에 유기 트래픽이 본격적으로 우상향  
  - 연구에 따르면, 대부분의 새 페이지가 의미 있는 상위권(Top 10)에 진입하는 데 <strong>6–12개월</strong>이 걸린다는 분석도 있음

실제 SEO 포럼들에서는:

- <strong>“새 사이트는 처음 2–3개월은 아예 신경 끄고 글이나 쓰라”</strong>  
- <strong>“체감 되는 트래픽은 반 년~1년은 깔고 간다”</strong> 같은 말이 반복된다.

즉, <strong>6–12개월 동안 검색 유입 0</strong>은  
신규 도메인 입장에서는 거의 기본 난이도라고 보면 된다.

<br/>

## 3. 이른바 ‘샌드박스 구간’은 어느 정도로 봐야 하나

![유리 상자 안에서 보호받으며 자라나는 어린 식물과 디지털 데이터 스트림](/images/new_domain_sandbox_seedling_1770944242037.png)

완전 과학적인 용어는 아니지만,  
많은 사람들이 <strong>“샌드박스(sandbox)”</strong> 라고 부르는 구간이 있다.

요약하자면:

- 새 도메인에 대해 구글이 <strong>“진짜 괜찮은 사이트인지”</strong> 보는 기간  
- 이 시기에는 색인은 되지만, 순위가 잘 안 오르거나 불안정하게 출렁임

각종 실전 사례를 보면:

- 어느 정도 경쟁 있는 니치에서는 <strong>3–6개월 정도</strong>를 샌드박스 구간으로 보는 의견이 많고
- 특히 돈 관련(YMYL)이나 스팸 많은 분야는 <strong>8–12개월까지도 길어질 수 있다</strong>는 얘기가 많다.

반대로:

- 경쟁이 거의 없는 니치, 로컬 비즈니스처럼 신뢰 얻기 쉬운 케이스는  
  1–2개월 안에도 어느 정도 유입이 생기는 사례가 꽤 있다.

정리하면,

- <strong>“샌드박스 = 6개월 고정”</strong> 이런 건 아니고  
- <strong>내 니치의 경쟁도 + 도메인 신뢰도</strong>에 따라  
  1–3개월에 끝날 수도, 12개월까지도 갈 수도 있다.

<br/>

## 4. “몇 개월까지 0이면 걱정해야 하나?” 구간별 기준

실전 감각으로 선을 한 번 그어보면:

- <strong>1개월 미만</strong>  
  - 유입 0은 거의 기본값, 걱정보다 “기술 체크 + 콘텐츠 쌓기” 단계  

- <strong>1–3개월</strong>  
  - 글이 20–30개 이상인데 검색 인상(impression)이 완전 0이면 기술 세팅 점검  
  - 하루 몇 클릭이라도 들어오기 시작했다면, 매우 정상적인 흐름  

- <strong>3–6개월</strong>  
  - 글 수가 50개 이상인데 여전히 검색 유입이 사실상 0이라면 전략 점검  

- <strong>6–12개월</strong>  
  - 이쯤까지도 유입이 0에 가깝다면 <strong>“전략 자체를 다시 짜야 할”</strong> 가능성이 크다.  

요약하면,

- <strong>“완전 새 도메인 + 2개월 유입 0” → 아직 크게 걱정할 타이밍은 아니다.</strong>  
- <strong>“6개월 이상 유입 0 근처” → 진짜로 구조·전략 점검이 필요하다.</strong>

<br/>

## 5. 0인 기간에 뭘 하고 있어야, 나중에 팍 튀는지

![정체기를 지나 급상향하는 마법 같은 서치 콘솔 그래프와 디자이너의 워크스페이스](/images/new_domain_momentum_graph_trend_v2_1770944357283.png)

그럼 0 구간에서 그냥 멍하니 시간만 보낼 거냐.  
그건 또 아니다.

현실적으로 효율 좋은 루틴은 대략 이렇다.

1) <strong>기술 베이스 정리</strong> (초기 1–2주)  
- Search Console 등록 + sitemap 제출  
- 기본 SEO 태그(title, description, OG) 세팅  

2) <strong>저경쟁 키워드 위주로 20–50개 글 쌓기</strong> (1–3개월)  
- 경쟁이 낮은 롱테일 키워드 위주, 실제 사례와 숫자 활용

3) <strong>내부 링크 구조 잡기</strong> (글 10개 넘는 시점부터)  
- 글들끼리 자연스럽게 서로 링크하여 고립 방지

4) <strong>소규모라도 외부 신호 만들기</strong> (병행)  
- 외부 사이트, 커뮤니티, SNS에서 자연스러운 링크 확보

이걸 꾸준히 돌리고 있으면,  
보통 3–6개월 사이 어느 시점에 <strong>“그래프가 한 번 툭 올라가는”</strong> 구간이 온다.

<br/>

## 6. 멘탈 관리용 현실 체크

데이터 쪽에서 나온 몇 가지 숫자만 기억해도 멘탈이 좀 편해진다.

- 한 연구에서, <strong>새로 만든 페이지가 1년 안에 구글 Top 10에 들어갈 확률은 1.7% 정도</strong>라는 결과가 있다.
- 대부분의 새 페이지는 <strong>6–12개월</strong>은 지나야 의미 있는 순위에 접근한다.
- <strong>“3개월 안에 뭔가 보이면 빠른 편”</strong>, <strong>“6개월쯤에 유입이 늘기 시작하면 아주 정상”</strong>

그러니까,

- 1–2개월째 유입 0이라고 해서 <strong>“사이트가 망했다”</strong>는 신호는 아니다.  
- 오히려 그 시기에 기반을 얼마나 잘 깔았냐가 이후 1년짜리 그래프 모양을 거의 결정한다.

<br/>

## 그냥 한 줄로 말하면

- 신규 도메인에서 글 쓰고도 검색 유입 0인 기간은 <strong>1–3개월 정도면 충분히 정상</strong> 구간이고,  
- <strong>3–6개월 사이</strong>에 슬슬 인상·클릭이 생기기 시작하면 오히려 잘 크는 편이다.  
- <strong>6개월이 넘어가도 거의 0</strong>이면, 그때는 <strong>“시간 문제”</strong>가 아니라 <strong>“전략 문제”</strong>일 가능성이 크다.

그래서 초반에는 성적표만 들여다보지 말고,  
글을 1개 더 쓰는 쪽으로 시간을 쓰는 게 마음도, 결과도 더 좋다.  
구글이 느린 거지, 내가 못해서만은 아니다.
